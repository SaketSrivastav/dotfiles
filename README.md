# dotfiles
## Setup
   ```git clone git@github.mv.usa.alcatel.com:saketkus/dotfiles.git```
<br/>   ```cd dotfiles```
<br/>   ```python sync_vim_plugin.py```
<br/>   ```cd $HOME```
<br/>   ```ln -s $HOME/dotfiles/vimrc .vimrc```
<br/>   ```ln -s $HOME/dotfiles/vim .vim```
<br/>   ```ln -s $HOME/dotfiles/rcfunctions .rcfunctions```
